# LifePilot AI - WhatsApp Demo

This is a functional web-based demo simulating the **LifePilot AI** WhatsApp interface. It showcases the core "Coffee Chat" scheduling flow.

## üöÄ Quick Start

### 1. Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)

### 2. Installation
Navigate to the `product` directory and install dependencies:
```bash
cd product
npm install
```

### 3. Run Development Server
Start the interactive demo:
```bash
npm run dev
```
The demo will be available at `http://localhost:3000`.
