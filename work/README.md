# Work Library

Task management directory for the AI startup.

```
work/
├── vision/         # CEO's vision documents
├── to-do/          # Tasks waiting to be started
├── in-progress/    # Tasks currently being worked on
├── qa/             # Tasks awaiting verification
├── done/           # Completed tasks (numbered: 001-task.md)
└── design/         # Design documentation
    ├── product-definitions/   # PM & Designer specs
    ├── architecture/          # CTO architecture docs
    └── execution-logs/        # Developer implementation logs
```

**Task naming**: `[priority]-[role]-[task-name].md` (e.g., `high-developer-user-auth.md`)

See [work_management.md](../.claud/work_management.md) for task format and rules.
See [task_lifecycle.md](../.claud/task_lifecycle.md) for complete workflow.
